Getting Started

  • There are several spots in the source marked CHANGE_ME, look at all of those and take the appropriate actions.
  • If you have a textual description of your puzzle solution, you can mark partials in answer-blue and the final answer in answer-red.
  • If your puzzle uses PuzzleJS, then you can copy your puzzle HTML into this file, use the designer to record your solution, and then copy the exported HTML back into this file. Also, add solution to the data-mode property.
  • PuzzleJS designer: Press Shift + Ctrl + Alt + F12. How to use the designer
